# Project Context

## 프로젝트 개요
이 프로젝트는 문선명 연구소 홈페이지 개발 프로젝트다. 연구소는 내년 1월 출범을 목표로 하고 있으며, 현재는 계획 및 홍보 단계에 있다.

## 운영 단계
1. 출범 전 약 7개월
- 홍보용 홈페이지로 사용
- 홍보 자료 게시판 운영
- 공지사항 게시
- 문의글 수집
- 참여 신청 및 후원 관심자 수집

2. 출범 이후
- 공식 연구소 홈페이지로 운영
- 연구소 소개, 연구 활동, 자료 아카이브, 활동 보고, 후원 안내 기능 확장
- 출범 전 게시글, 문의글, 참여 신청 데이터는 삭제하지 않고 유지

## 개발자 상황
- CS 전공자
- 1인 개발 및 운영 예정
- React Native 웹앱 개발 경험 있음
- Python FastAPI로 AI 모델을 로컬 서빙한 경험 있음
- DB 운영 경험은 없음
- MacBook에서 개발
- Codex를 활용해 개발 예정

## 기술 방향
- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- 초기 로컬 DB는 Docker PostgreSQL 사용
- 장기적으로 AWS 배포 고려
- 초기 배포는 AWS Amplify 또는 단순한 AWS 구조를 우선 고려

## 개발 원칙
- 1인 개발자가 유지보수 가능한 구조 우선
- 처음부터 과도한 커뮤니티 기능을 만들지 않음
- 게시글과 문의글은 장기 보존을 전제로 설계
- 출범 전 홍보 사이트가 출범 후 공식 홈페이지로 자연스럽게 전환될 수 있어야 함
- URL과 데이터 구조는 나중에 버리지 않을 형태로 설계
- 관리자 기능은 최소 기능부터 구현

## 초기 MVP 기능
- 메인 페이지
- 연구소 소개
- 설립 취지
- 공지사항
- 홍보자료 게시판
- 문의/참여 신청 폼
- 관리자 로그인
- 관리자 게시글 작성/수정/삭제
- 게시글 카테고리 관리
- 파일 업로드는 초기에는 보류 가능

## 게시판 방향
초기에는 완전한 자유게시판보다 관리형 게시판을 우선한다.

방문자 기능:
- 공지사항 보기
- 홍보자료 보기
- 문의 남기기
- 참여 신청하기
- 후원 관심 등록하기

관리자 기능:
- 공지 작성
- 홍보자료 작성
- 게시글 수정/삭제
- 문의 확인
- 참여 신청자 관리

## 향후 확장
- 댓글

- 후원 시스템
- 뉴스레터
- 관리자 권한 분리
- S3 파일 업로드
